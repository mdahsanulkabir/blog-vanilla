HTML: HyperText Markup Language

HTML is the basic building block of the web. It defines the skeleton and the meaning of web content. Other technologies are used with HTML to describe the appearance/presentation and functionality of the web contents.

From MDN's website we get the idea of Hypertext like below:

"Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.

HTML uses "markup", which includes special elements, which are consists of tags.

Therefore, HTML is the links of annotated texts, images and other contents within single website or between websites.

Elements

We know that HTML uses "markup" to annotate text, images and other contents for display in web browser. HTML markup includes special elements like "<head>", "<p>", "<img>" etc.

Let's checkout how an element is formed

Structure of an Element

Below picture taken from MDN website

From the above picture, we can see that an element has several parts.

  • Opening tag - or, start tag. It represents the name of the tag
  • Closing tag - or, end tag. Its same as opening tag followed by forward slash
  • Content - It stays between the two tags
    • Few elements have own closing tags. I.e., they don't have ending tags. These are called void element.

Nesting of elements

An element can be placed within other elements. For example check below box.

I am very excited to learn coding.

        <p>I am very excited to learn coding</p>
The whole thing in the box (including) is inside a block named 'div'

If we want to make the word 'very' bold, we can use another element inside this p element. 

        <p>I am <strong>very</strong> excited to learn coding</p>

I am very excited to learn coding.

Therefore, nesting an element inside another element is possible. And it is required in many instances to reflect desired output.

Attributes of Elements

Elements might have attributes with following syntax.

          <p title="This is a tooltip">I Love Javascript.</p>

I Love Javascript.

'Title', as an attribute of an element provides a tooltip to show information on hover on the element.

Boolean Attributes

Sometimes attributes have no value. Those are actually boolean types of attributes. Boolean type attributes have generally single value, which is generally same as the attribute name.

Example:

 
          <input type="text" disabled="disabled">

Details of head element

The head element is not displayed in the web browser. It contains information such as title, links to css, links to favicons, some other metadata. Browser uses these information to render the HTML document correctly.

<title></title>

- This should not be confused with the h1 element which can serves as the title of the content. The title element inside the head tag serves as a metadata about the document itself. It is visible at the tab of the brower and while we bookmark that page, this title tag content serves as the name of the bookmark. 

<meta></meta>

- The meta tags provide other informations for the browsers like below.

Some common meta tags

Usually meta data written as pair of name or property and a value

<meta charset="utf-8" />

- This metadata provides the information about the characters used in the document. 

<meta name="author" content="John Doe" />

- this metadata provides the author information of the page. 

<meta name="owner" content="Mohammed" />

- this metadata provides the owner information of the page. 

<meta name="description" content="This is a sample website" />

- this metadata provides the description of the page. It is very important for the SEO. When we search webpage in search engine, this description will be presentated below the title and link of the document.

favicons

favicon is the short for 'favorites icons'. Typically it is a square picture of 16-pixel square icon. It is used generally in brower tab, and next to bookmarked page in bookmarks panel.

It is saved as a .ico format in the same directory as site's index page. .gif or .png formats also support as favicon.

It is added in the <head> block as nested element as below

<link rel="icon" href="favicon.ico" type="image/x-icon" />

link tags for css and script tags for Javascript will be discussed later.
