Javascript

JavaScript (JS) is a lightweight interpreted (or just-in-time compiled) programming language with first-class functions.

JavaScript is a prototype-based, multi-paradigm, single-threaded, dynamic language, supporting object-oriented, imperative, and declarative (e.g. functional programming) styles.

How to add JavaScript to the page?

We can add JavaScript in two ways as below.

  • Internal Javascript -

    We can add JavaScript code with a <script></script> element just before closing of the </head> tag.

    • 
                    <head>
                        ...
                        <script>
                            // JavaScript goes here
                        </script>
                    </head>
  • External JavaScript - We can save our JavaScript code in separate file with .js extension and use the script tag with a src property mentioning the link of the file.

    .

    • 
                    <head>
                        ...
                        <script src="script.js" defer></script>
                    </head>
  • Inline JavaScript handlers - some times JavaScript is written inside html like below.
    •  Javascript
                    function createParagraph() {
                        const para = document.createElement("p");
                        para.textContent = "You clicked the button!";
                        document.body.appendChild(para);
                    }
                
                HTML
                    <button onclick="createParagraph()">Click me!</button>
    However, this is a bad practice. We should use addEventListener in this case as below. 
    
                    const buttons = document.querySelectorAll("button");
                    
                    for (const button of buttons) {
                    button.addEventListener("click", createParagraph);
                    }
