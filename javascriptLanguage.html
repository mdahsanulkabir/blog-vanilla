Variable

In any language, we have to deal with data, store the data, use the data also delete and modify the data. To deal with data, we have to store it in some place. In JavaScript we store the data in Variables (which uses memory locations and use bits to store the data, but that is beyond our scope for now).

Variable Declarations

In JavaScript, we have three kinds of variable declarations.

  • var - Declares a variable, optionally initializing it to a value.
  • let - Declares a block-scoped, local variable, optionally initializing it to a value.
  • const - Declares a block-scoped, read-only named constant.

Identifiers

Varibles are used by assigning a symbolic name. The name of the variable called Identifier

In JavaScript identifier

  • usually starts with a letter, underscore (_), or dollar sign ($).
  • Subsequent characters can also be digits (0 - 9).
  • Because JavaScript is case sensitive, letters include the characters A through Z (uppercase) as well as a through z (lowercase).
