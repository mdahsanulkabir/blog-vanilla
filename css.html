Tutorial on CSS

Basics of CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Like HTML, CSS is not a programming language. It is a style sheet language used to style the HTML elements.

For example - below is an example of css ruleset.


                p {
                    color: red;
                }

This above code can be used to change the color of the text of a paragraph in to red.

Here, p represents the selector, "color: red;" represents the declaration (where, "color" is the property, and "red" is the property value). We can have multiple declarations under a selector.

Therefore, we can say that CSS ruleset is a combination of 'selector' and 'declaration'.

Within each declarations we need to use semicolon (;). Check below code.


                p {
                    color: red;
                    width: 500px;
                    border: 1px solid black;
                }

To style multple elements with same ruleset, we can separate multiple selectors with comma (,).


                p,
                li,
                h1 {
                    color: red;
                }

CSS syntax

CSS is a rule based language. We define the rules by spcifying groups of styles that are applied to particular elements or groups of elements on the web page.
Note: When multiple groups implement rules on an element, specificity come in to place and define which rule should be applied. Specificity will discussed later.

Example:
            h1 {
                color: red;
                font-size: 5em;
            }
                
            p {
                color: black;
            }

Above code presents a stylesheet of two styles. Where, the first style makes all the h1 elements (however, a document supposed to have only one h1 header element) in red color and font size as 5 rem. Second style applies to all paragraph elements and makes those elements in to black color.

CSS Structure

Method of styling

We can apply to our document in three ways as below.

  1. External stylesheet
  2. Internal stylesheet
  3. Inline styles

External Stylesheet

External stylesheet contains the css rules in a separate file with .css extension. This is the most common method to apply the css in our document. We have to use <link> element nested in the <head> element in our document. This external stylesheet can be located in our folder structure or in 3rd party source server.

referencing a stylesheet file from our folder structure and linking to our document can be like below.


                    <!doctype html>
                    <html lang="en-GB">
                    <head>
                        <meta charset="utf-8" />
                        <title>My CSS experiment</title>
                        <link rel="stylesheet" href="styles.css" />
                    </head>
                    <body>
                        <h1>Hello World!</h1>
                        <p>This is my first CSS example</p>
                    </body>
                    </html>

Here the name of the file is 'styles.css' and it is located in the same directory as the document.

Generally, we put the stylesheet files in a separate folder named 'styles'. Thus we will link that file to our document as below.


                    <!doctype html>
                    <html lang="en-GB">
                    <head>
                        <meta charset="utf-8" />
                        <title>My CSS experiment</title>
                        <link rel="stylesheet" href="styles/styles.css" />
                    </head>
                    <body>
                        <h1>Hello World!</h1>
                        <p>This is my first CSS example</p>
                    </body>
                    </html>

Associated rules in that stylesheet file may look like below.


                    /*  styles/styles.css */
                    h1 {
                        color: blue;
                        background-color: yellow;
                        border: 1px solid black;
                    }
                    
                    p {
                        color: red;
                    }

Internal stylesheet

In this method of style, the css rules are placed as nested in the <head, element of the document and wrapped with <style> tags as below.


                    <!doctype html>
                    <html lang="en-GB">
                    <head>
                        <meta charset="utf-8" />
                        <title>My CSS experiment</title>
                        <style>
                            h1 {
                                color: blue;
                                background-color: yellow;
                                border: 1px solid black;
                            }
                            
                            p {
                                color: red;
                            }
                        </style>
                    </head>
                    <body>
                        <h1>Hello World!</h1>
                        <p>This is my first CSS example</p>
                    </body>
                    </html>

Inline style

In inline style method, the rules are applied directly with the 'style' property of an element. In type of style the 'selector' is not needed as the rule is directed applied to the element. Below is an example of inline style that has same effect as the earlier example.


                    <!doctype html>
                    <html lang="en-GB">
                    <head>
                        <meta charset="utf-8" />
                        <title>My CSS experiment</title>
                        <style>
                            h1 {
                                color: blue;
                                background-color: yellow;
                                border: 1px solid black;
                            }
                            
                            p {
                                color: red;
                            }
                        </style>
                    </head>
                    <body>
                        <h1 style='color: blue; background-color: yellow; border: 1px solid black;'>Hello World!</h1>
                        <p style='color: red;'>This is my first CSS example</p>
                    </body>
                    </html>

All the above three examples will render the document. Like below

Hello World!

This is my first CSS example

Style a document with inline style method is least efficient. Therefore, we will try to style our document with other style methods (where the external style method is most preferred)

Selector

If we style our document with external styling method or internal styling method, we have to use selector. Selector targets the HTML element to style its content. Each CSS rule starts with a selector, or a list of selectors. Below are some valid CSS selectors


                h1
                a:link
                .manythings
                #onething
                *
                .box p
                .box p:first-child
                h1, h2, .intro

Specificity

Previously, I mentioned that there might me some occations where two selectors may select same HTML element. Below is an example.


                .special {
                    color: red;
                }
                
                p {
                    color: blue;
                }

With an element like below


                <p class="special">What color am I?

Above, we have a paragraph with a class property named 'special'. p and .special selector sets the text color of the p element differently. Here comes the concept of specificity. Specificity helps us to solve this conflict. CSS has rules to control which selector is stronger in the event of such conflicts. These rules are called cascade and specificity. Lets render the paragraph.

What color am I?

The element is styles with red color because the class has more specificity value than the paragraph in this context

Frustration - Sometimes, CSS might not apply as we expected because something else in the stylesheet has more specificity. Recognizing that more than one rule could apply to an element is the first step in fixing these kinds of issues.

Properties and values

Properties and associated values are the most basic components of CSS. Pair of property and value is called declaration. Multiple declaration applied in a selector is called declaration block. Finally the whole rule including the selector is called CSS Ruleset or CSS rule.

Note: If a declaration is invalid, then the browser ignore the declaration without any warning or error.

Shorthands

Some properties like 

font, background, padding, border, and margin
are called shorthand properties. This is because shorthand properties set several values in a single line.

For example, below one line code - 


                padding: 10px 15px 15px 5px;

correspons to following four lines of code.


                padding-top: 10px;
                padding-right: 15px;
                padding-bottom: 15px;
                padding-left: 5px;

Another example:


                background: red url(bg-graphic.png) 10px 10px repeat-x fixed;

Same as..


                background-color: red;
                background-image: url(bg-graphic.png);
                background-position: 10px 10px;
                background-repeat: repeat-x;
                background-attachment: fixed;
